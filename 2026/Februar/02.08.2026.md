## 1. Always with ACCUSATIVE
| Preposition | Meaning (EN) |
|-------------|--------------|
| durch | through |
| für | for |
| gegen | against |
| ohne | without |
| um | around / at (time) |
| bis | until / up to |

## 2. Always with DATIVE
| Preposition | Meaning (EN) |
|-------------|--------------|
| aus | out of / from (inside) |
| außer | except for |
| bei | at / near / at someone's place |
| mit | with |
| nach | to / toward / after (time) |
| seit | since / for (time duration) |
| von | from / of |
| zu | to / toward |

---

## 3. Two-way prepositions (ACCUSATIVE or DATIVE)

**ACCUSATIVE → movement (where to?)**  
**DATIVE → position (where?)**

| Preposition | Meaning (EN) |
|-------------|--------------|
| an | at / on (vertical surface, edge) |
| auf | on / onto (horizontal surface) |
| hinter | behind |
| in | in / into |
| neben | next to |
| über | above / over |
| unter | under / below |
| vor | in front of / before |
| zwischen | between |

---

## Examples

**Accusative (movement):**
- Ich gehe **in die** Schule.  
- Ich lege das Buch **auf den** Tisch.

**Dative (position):**
- Ich bin **in der** Schule.  
- Das Buch liegt **auf dem** Tisch.
